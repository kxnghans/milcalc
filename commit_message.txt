feat(mobile): improve keyboard handling and UI

This commit introduces several improvements to the mobile app:

* Replaces `KeyboardAvoidingView` with `react-native-keyboard-aware-scroll-view` for better keyboard handling.
* Adds a `DismissKeyboardView` component to dismiss the keyboard when tapping outside of an input.
* Improves the `TwoColumnPicker` modal to be dismissable.
* Updates the splash screen with seasonal effects.
* Refactors the PT calculator to use `StrengthComponent`, `CoreComponent`, and `CardioComponent`.
* Enhances `SegmentedSelector` to allow custom styling for selected segments.
